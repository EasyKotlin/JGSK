# JGSK
本项目 是[JGSK](http://git.bookislife.com/topics/jgsk/) 的示例代码。项目使用Maven进行构建，包名的序号与博客标题的序号一致。
主要分章节列举了Java,Groovy,Scala,Kotlin 四种语言各自的特点。
